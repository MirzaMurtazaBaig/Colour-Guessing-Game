# Colour-Guessing-Game
A fun color guessing game with GUI - guess the computer's color!
Requirements Python 3.7+  tkinter (comes with Python)  Game Controls Color Buttons - Make your guess  New Game - Reset the game  Quit - Exit the game  Features 🎨 Colorful interface  💫 Victory animations  📊 Real-time scoring  🔄 New game option 
Code Structure
ColorGuessingGame Class
├── __init__() - Setup game
├── setup_gui() - Create interface
├── new_round() - Start new round
├── make_guess() - Check guess
└── game_over() - End game
